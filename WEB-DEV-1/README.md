## ğŸ“˜ Detailed Syllabus

### ğŸ§¾ Unit 1: HTML & Web Foundation  
**Hours:** 10  

**Content:**  
- Introduction to Web and HTML5  
- Structure: `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`  
- Basic tags: headings, paragraphs, line breaks, horizontal rules  
- Links: external, internal, mailto, download  
- Images: `<img>`, responsive setup, alt text  
- Lists: ordered `<ol>`, unordered `<ul>`, definition lists `<dl>`  
- Tables: `<table>`, `<tr>`, `<th>`, `<td>`  
- Forms: `<form>`, `<input>`, `<select>`, `<textarea>`, `<button>`  
- Form attributes: `placeholder`, `required`, `type`, `action`, `method`  
- Semantic HTML5 tags: `<header>`, `<footer>`, `<section>`, `<article>`, `<nav>`  
- Accessibility (ARIA basics) and SEO-friendly structure  

**Practice Tasks:**  
- Create a personal introduction page  
- Build a feedback form using labels and inputs  
- Design a simple portfolio with internal navigation  

---

### ğŸ¨ Unit 2: CSS â€“ Core & Advanced  
**Hours:** 15  

**Content:**  
- CSS basics: Inline, internal, external stylesheets  
- Syntax, selectors, and specificity rules  
- Colors (HEX, RGB, HSL), gradients, and shadows  
- Fonts and typography with Google Fonts & `@font-face`  
- Box model: margin, padding, border, and content  
- Display types: inline, block, inline-block, flex, grid  
- CSS position properties: static, relative, absolute, fixed  
- Flexbox: direction, justify-content, align-items, wrap  
- Grid layout: columns, rows, grid areas  
- Responsive design principles (mobile-first, breakpoints)  
- Media queries (`@media screen and (max-width: 768px)`)  
- CSS transitions, transformations, and keyframe animations  
- Hover, focus, and active states  
- Debugging with Chrome DevTools  

**Practice Tasks:**  
- Create a responsive layout using Flexbox  
- Animate navigation links with hover transitions  
- Design a three-section responsive portfolio homepage  

---

### âš™ï¸ Unit 3: JavaScript Essentials  
**Hours:** 10  

**Content:**  
- Introduction to JavaScript and browser execution  
- Setting up `<script>` in HTML and external JS files  
- Variables and constants: `var`, `let`, `const`  
- Data types: string, number, boolean, object, array, null, undefined  
- Operators: arithmetic, assignment, comparison, logical  
- Conditional statements (`if`, `else`, `switch`)  
- Loops: `for`, `while`, `do-while`  
- Functions: declaration, arrow, anonymous, default parameters  
- Arrays and objects: CRUD operations, iteration methods (`map`, `filter`, `reduce`)  
- DOM Manipulation:  
  - Accessing elements (`getElementById`, `querySelector`)  
  - Changing styles, text, and attributes dynamically  
  - Event listeners (`onclick`, `addEventListener`)  
- Basic form validation using JavaScript  
- Alert, prompt, and confirm dialog interactions  

**Practice Tasks:**  
- Create a simple interactive quiz app  
- Build a dynamic image slider  
- Validate form input before submission
