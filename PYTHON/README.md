- Understanding Python‚Äôs ecosystem: CPython, Miniconda, VS Code setup

### **Unit 2: Control Flow, Functions & Data Structures**
**Hours:** 12  
**Content:**
- Conditionals (`if`, `elif`, `else`) and nested logic

- Functions: arguments, return values, scope, type hints

- Built-in data structures:

- Algorithmic idioms:
  - Frequency maps, sliding window, two-pointer methods  
- Basic Big-O complexity understanding  

**Practical Focus:**  
- Writing modular reusable functions

### **Unit 3: Object-Oriented Programming, Modules, and Robust Code**

**Content:**

- File handling (text, CSV, JSON) using `pathlib` and `open()`  
- Exception handling:
  - `try`, `except`, `else`, `finally`  
  - Custom exception classes and logging  
- Packaging and modularization:
  - Creating and importing modules  
  - Folder structure and `__init__.py` usage  
- Introduction to testing frameworks: `unittest` and `pytest`  
- Continuous integration basics using GitHub Actions
