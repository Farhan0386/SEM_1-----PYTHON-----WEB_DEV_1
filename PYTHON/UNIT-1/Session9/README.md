List comprehensions are a Pythonic way to create new lists using a single line of code.
They combine looping, filtering, and transformation into a compact syntax.

ðŸ”¹ Why Use Them?

- Cleaner and more readable than traditional loops
- Often faster in execution
- Useful for data transformation, filtering, and mapping
- Supports arithmetic, string, and conditional logic

ðŸ”¹ Syntax
[expression for item in iterable if condition]

ðŸ”¹ Example
squares = [x**2 for x in range(5)]

Creates: [0, 1, 4, 9, 16]
