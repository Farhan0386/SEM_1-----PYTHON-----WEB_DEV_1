5# --Python Keywords -Python keywords are reserved words that are used to
define the syntax and structure of the Python 
  
   if: Conditional statement for executing code based on a condition.
   else: Executes when the if condition is false.
   elif: Short for ‚Äùelse if‚Äù, used for multiple branching in conditions.
   for: Used for looping over a sequence (e.g., list, tuple, string).
   while: Loops until a condition becomes false.
   def: Defines a function.
   return: Exits a function and returns a value.
   class: Defines a new class.
   import: Imports a module into the current script or module.
   try, except: Used for exception handling.
